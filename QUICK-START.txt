â“ TROUBLESHOOTING
-----------------

Problem: "npm is not recognized"
Solution: Install Node.js first (Step 1)

Problem: "Port 3000 is already in use"
Solution: 
  - Open server.js file
  - Change line 12: const PORT = 3000;
  - To: const PORT = 8080;
  - Save file and restart

Problem: Nothing happens when I run npm start
Solution:
  - Make sure you're in the correct folder
  - Check terminal is at: tenant-management-app
  - Try: npm install (then npm start again)
